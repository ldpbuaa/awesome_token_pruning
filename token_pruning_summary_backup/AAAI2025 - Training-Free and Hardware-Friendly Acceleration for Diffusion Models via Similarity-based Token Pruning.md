# 论文深度总结：Training-Free and Hardware-Friendly Acceleration for Diffusion Models via Similarity-based Token Pruning

## 1. 研究动机

扩散模型在图像生成领域取得了显著突破，推动了图像到图像翻译、文本到图像生成、视频生成和图像编辑等多样化下游应用的发展。然而，扩散模型的卓越性能总是伴随着巨大的计算成本，这限制了它们在边缘设备和交互式应用中的部署。现有加速方法主要关注减少采样步骤和压缩去噪网络，而扩散模型输入数据的大小作为计算复杂度的重要贡献者尚未得到充分研究。此外，现有的针对图像生成的token减少方法存在局限性：ToMeSD直接应用分类任务的token减少方法未考虑扩散模型的生成特性；AT-EDM利用图算法选择冗余token，但收敛时间不可控，对硬件不友好，不支持批量计算，缺乏实际应用价值。

## 2. 解决问题

论文致力于解决的核心科学问题是：如何通过减少输入数据中的冗余token来加速扩散模型的计算过程，同时保持甚至提高生成质量。具体挑战包括：
- 在生成任务中，被修剪的token需要被恢复（与分类任务不同）
- 最小化恢复误差，即被修剪token与用于恢复它们的token之间的差异
- 确保token修剪策略在扩散模型的多个去噪步骤中保持平衡，避免某些token被过度修剪
- 开发一种对硬件友好、无需训练且不校准数据的实用方法

## 3. 现象分析

作者观察到以下关键现象：
- 扩散模型在不同时间步的token表示非常相似，导致相邻时间步中选择的基础token和修剪token可能高度重复
- 由于被修剪token通过直接从最相似的基础token复制来恢复，这些token在后续去噪步骤中倾向于保持与基础token的高相似性，导致它们在几乎所有时间步都可能被修剪
- 这种极度不平衡的token修剪会导致生成质量显著下降
- 图像中同一区域的块携带相似信息，意味着空间维度上相邻的token可能有相似的表示
- 在相邻时间步中，相同token被重复修剪的比例高达97%，需要通过引入随机性来降低这一比例

## 4. 主要方法

论文提出了**SiTo (Similarity-based Token Pruning)**，一种三阶段的token修剪方法：

### 基础token选择
1. **余弦相似度计算**：计算所有token之间的余弦相似度矩阵 CosSim ∈ R<sup>B×N×N</sup>
2. **相似度分数(SimScore)**：计算每个token与其他token的相似度总和 SimScore<sub>b,i</sub> = Σ<sub>j</sub> CosSim<sub>b,i,j</sub>
3. **随机性选择**：添加高斯噪声到SimScore中引入随机性，避免相邻时间步重复选择相同token
4. **基于块的token选择**：将图像划分为s×s的非重叠块，在每个块中选择具有最高SimScore的token作为基础token，确保基础token在空间上均匀分布

### 修剪token选择
- 基于选定的基础token，选择与基础token最相似的token进行修剪
- 修剪的token数量由修剪比例K控制

### 修剪token恢复
- 将未修剪的token输入到神经层进行计算
- 通过直接从最相似的基础token复制来恢复被修剪的token

SiTo的核心机制是通过选择与所有其他token相似度最高的token作为基础token，并选择与基础token最相似的token进行修剪，从而最小化恢复误差。

## 5. 数据与实验

### 数据集
- **ImageNet-1k**：生成2,000张图像（每类2张）
- **COCO30k**：生成30,000张图像（每个描述1张）

### 基线模型
- **ToMeSD**：基于分类任务的token合并方法
- **AT-EDM**：基于图算法的token修剪方法

### 核心实验结果
- **ImageNet**：SiTo实现1.90倍加速和2.70倍内存压缩，同时FID降低1.33
- **COCO30k**：SiTo实现1.75倍加速，同时FID降低1.15
- **生成质量**：SiTo优于ToMeSD，特别是在处理复杂提示时，生成的图像更真实、细节更丰富，与原始图像和文本提示的匹配度更高
- **兼容性**：SiTo能够与现有的加速方法（如快速采样器）良好配合
- **硬件测试**：在单4090 GPU上进行评估，测量了平均延迟和加速比

## 6. 主要贡献

这篇论文的主要贡献包括：

1. **新方法SiTo**：提出了首个基于相似性的token修剪方法，用于加速扩散模型而不牺牲生成质量

2. **基础token概念**：引入了"base tokens"的创新概念，这些token在修剪token的选择和恢复中起着关键作用

3. **三个关键设计原则**：
   - **最大相似性**：选择与所有其他token相似度最高的token作为基础token
   - **均匀空间分布**：确保基础token在图像的不同空间位置上均匀分布
   - **随机选择**：通过添加高斯噪声引入随机性，将相邻时间步中相同token被重复修剪的比例从97%降低到72%

4. **无需训练且数据无关**：SiTo是非参数化的，不需要训练，也不需要任何验证数据

5. **硬件友好**：计算成本低，无额外内存占用，支持CUDA和并行计算

6. **实际应用价值**：即插即用，可直接用于现实世界的应用场景，与现有的加速方法兼容
