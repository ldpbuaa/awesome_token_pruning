# 论文总结：TOKEN PRUNING MEETS AUDIO: INVESTIGATING UNIQUE BEHAVIORS IN VISION TRANSFORMER BASED AUDIO CLASSIFICATION

## 1. 研究动机

作者进行这项研究的主要动机是探索Vision Transformer (ViTs)在音频分类任务中应用token pruning技术的独特行为。具体背景和痛点包括：

- ViTs已在各种计算机视觉任务中取得state-of-the-art性能，但高计算成本阻碍了其实际应用
- token pruning技术被提出用于减少ViTs的计算负载，在视觉任务中通过丢弃非物体区域标记来有效工作
- 然而，将token pruning应用于音频任务面临独特挑战：在音频处理中，区分相关与不相关区域不如视觉任务直观
- 对于音频分类任务，尚不清楚哪些标记应该被保留或丢弃，因为Mel频谱图中的背景(低强度)区域不一定缺乏信息
- 例如，在婴儿哭声分类中，丢失哭声之间的短暂静音可能使其难以与警报声区分；某些频率带的空白区域也可能是声源的特征

## 2. 解决问题

论文致力于解决以下核心科学问题和技术难题：

- 确定在音频分类中哪些标记应该被保留或丢弃，这与图像分类存在根本差异
- 分析token pruning在音频任务中的独特行为模式
- 探索如何在保持高准确率的同时显著减少计算量(实现2×的MAC操作减少)
- 理解信号区域(high-intensity)和背景区域(low-intensity)标记在音频分类中的不同角色
- 研究基于注意力的token pruning方法在音频任务中的有效性
- 探索不同预训练模型(AudioMAE与AST)在token pruning行为上的差异

## 3. 现象分析

作者观察到几个关键现象：

1. **与图像分类截然不同的标记剪枝模式**：
   - 在图像分类中，token pruning倾向于保留与标签相关的标记，丢弃背景组件标记(如图1所示)
   - 在音频分类中，token pruning模式完全不同，既会剪枝信号标记，也会剪枝背景标记(如图2所示)
   - 在某些情况下，大部分信号标记被剪枝，这表明音频分类与图像分类存在根本差异

2. **背景标记的重要性**：
   - 背景区域(低强度)的标记经常被保留，而信号区域(高强度)的标记反而经常被剪枝
   - 这一现象表明模型依赖于背景区域的信息，与图像分类中通常丢弃背景的做法相反

3. **标记保留比例**：
   - 随着keep-rate(保留率)降低，保留的信号标记比例也随之降低
   - 例如，当keep-rate为0.5时，只有约33%的信号标记被保留(表2)

4. **注意力分数分析**：
   - 保留的标记比被剪枝的标记具有显著更高的注意力分数(表3和图6)
   - 随着keep-rate增加，模型对其标记选择更有信心

5. **残差连接的影响**：
   - 在较后的块中，自注意力层对模型分类结果的影响更大(图7)
   - 这与图像分类中的趋势相反，在图像分类中，较后层的残差连接影响更大

## 4. 主要方法

论文提出了AudioMAE-TopK模型，其核心机制包括：

1. **模型架构**：
   - 基于AudioMAE(一种基于ViT的自监督音频模型)作为骨干网络
   - 采用TopK方法作为token pruning技术，根据token-to-token注意力分数选择要保留的标记
   - 在第4、7和10个ViT块之间插入token pruning模块
   - 每个块中，保留注意力分数最高的N×keep-rate个标记，其余被剪枝

2. **标记分类方法**：
   - 使用阈值τ区分不同数据集的信号和背景标记(基于Mel频谱图的强度值)
   - 例如，对于SPC-2数据集，τ设为-0.65；对于ESC-50，τ同样设为-0.65

3. **注意力分数计算**：
   - 使用token-to-token注意力分数的平均值来评估每个标记的重要性(公式2a)
   - 对比实验中使用了基于[CLS]标记的注意力分数(公式2b)

4. **训练策略**：
   - 在微调阶段不使用掩码策略(包括SpecAug)，因为减少标记的训练已经施加了大量掩码
   - 使用层级学习率衰减，特别是对于ESC-50数据集采用更高的最小学习率(10^-5)

## 5. 数据与实验

1. **数据集**：
   - Speech Commands V2 (SPC-2)：35个常见语音命令的84,843个训练样本
   - Environmental Sound Classification (ESC-50)：50个类别的2,000个5秒音频样本
   - VoxCeleb-1：150K音频样本，用于识别1,251个不同说话者
   - AudioSet任务(AS-20K)：21K片段和19K评估片段

2. **基线模型**：
   - AudioMAE：作为骨干网络和主要基线模型
   - AST-TopK：用于比较不同预训练模型的token pruning行为

3. **核心实验结果**：
   - **性能与计算权衡**：
     - AudioMAE-TopK模型可以在计算量(MAC)减少2倍的情况下，保持准确率下降不到1%
     - 对于SPC-2，当keep-rate为0.5时，准确率从97.95%下降到97.28%(下降0.67%)，MAC从5.61G减少到2.81G
     - 对于ESC-50，当keep-rate为0.5时，准确率从94.30%下降到93.40%(下降0.90%)，MAC从23.11G减少到11.37G

   - **标记保留模式**：
     - 当keep-rate为0.5时，只有约33%的信号标记被保留(表2)
     - 背景标记保留比例较高，约为50%(表2)

   - **注意力分数分析**：
     - 保留的标记比被剪枝的标记具有显著更高的注意力分数(Γ值普遍大于1，表3)
     - 随着keep-rate增加，模型对其标记选择更有信心(Γ值增大)

   - **消融研究**：
     - 强制模型在剪枝阶段只保留信号标记会导致准确率显著下降，表明背景标记包含独特且不可替代的信息
     - AST-TopK模型比AudioMAE-TopK保留更多的信号标记

## 6. 主要贡献

这篇论文对该领域的最大贡献包括：

1. **开创性研究**：据作者所知，这是首次将token pruning应用于基于ViT的音频分类任务，并提供了对剪枝模式的深入分析

2. **发现音频与视觉任务中的token pruning根本差异**：
   - 在音频分类中，背景标记经常被保留，而信号标记反而经常被剪枝
   - 这一发现挑战了传统假设，即信号区域总是比背景区域更重要
   - 作者发现背景区域(低强度区域)包含独特、不可替代的信息，对于AudioMAE模型的性能至关重要

3. **提出高效模型**：AudioMAE-TopK模型实现了计算量减少2倍的同时，准确率下降不到1%，为音频分类任务提供了一种高效解决方案

4. **深入分析贡献**：
   - 揭示了信号标记和背景标记在音频分类中的不同角色
   - 研究了不同keep-rate下的标记保留模式
   - 探索了注意力分数与标记保留之间的关系
   - 通过残差连接分析解释了为什么许多信号标记可以被剪枝而不损失准确率

5. **实际应用价值**：该研究为在资源受限设备上部署音频分类模型提供了实用解决方案，同时保持了高准确率
