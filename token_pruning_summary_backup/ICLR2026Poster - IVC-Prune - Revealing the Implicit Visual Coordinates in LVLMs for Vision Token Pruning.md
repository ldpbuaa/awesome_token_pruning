# IVC-PRUNE: REVEALING THE IMPLICIT VISUAL COORDINATES IN LVLMs FOR VISION TOKEN PRUNING 论文总结

## 1. 研究动机

作者进行这项研究的主要动机是解决大型视觉-语言模型(Large Vision-Language Models, LVLMs)在处理高分辨率视觉输入时面临的**推理成本过高**问题。虽然现有的视觉标记剪枝方法已经取得了一定进展，但这些方法主要关注**语义相关性**，往往丢弃了对**空间推理**至关重要的标记。作者观察到，在视觉定位和空间推理任务上，现有剪枝方法的性能会显著下降，表明它们没有保留对空间理解至关重要的视觉标记。这种局限性严重阻碍了LVLMs在实际应用中处理高分辨率图像的能力。

## 2. 解决问题

论文致力于解决的核心科学问题是：如何在**减少视觉标记数量**的同时，保持LVLMs的**空间推理能力**。具体来说，作者需要解决以下技术难题：
1. 识别哪些视觉标记对空间推理至关重要
2. 设计一种剪枝策略，能够保留这些关键标记以及语义相关的标记
3. 确保剪枝过程不需要重新训练，并且能够适应不同的提示词

## 3. 现象分析

作者通过理论分析发现了一个关键现象：LVLMs通过**Rotary Position Embeddings (RoPE)** 隐式地建立视觉坐标系。具体现象包括：
- 当RoPE的旋转矩阵近似**单位矩阵**或**90°旋转矩阵**时，自注意力机制会隔离查询的绝对位置分量
- 特定的标记位置充当空间参考点：**实轴参考**（单位变换）和**虚轴参考**（90°旋转）
- 这些参考标记形成**隐式视觉坐标**(Implicit Visual Coordinates, IVC)，对于空间推理至关重要
- 现有剪枝方法仅保留语义相关的"前景"标记，导致空间推理任务性能下降

作者还发现，早期层剪枝导致性能下降的现象并非剪枝操作本身造成，而是由于意外移除了IVC标记。

## 4. 主要方法

论文提出了**IVC-Prune**，一种训练-free、prompt感知的剪枝策略，核心机制包括：

**IVC标记选择**：
- 基于RoPE的数学特性，通过计算每个标记位置的坐标得分来识别IVC标记
- 实轴得分 $V(m) = \sum_{k=0}^{d/2-1} \cos(m\theta_k)$，用于识别近似单位变换的位置
- 虚轴得分 $U(m) = \sum_{k=0}^{d/2-1} \sin(m\theta_k)$，用于识别近似90°旋转的位置
- 保留每种得分的前 $k_c$ 个标记

**前景标记选择**（两阶段过程）：
1. **语义种子识别**：使用值向量(value vectors)计算文本与视觉标记之间的注意力相似度，选择前1%的标记作为种子
2. **上下文前景细化**：使用文本标记和初始种子标记作为查询，扩展前景标记选择

**剪枝策略**：
- 在选定的中间层进行单次标记选择，保留原始位置ID
- 将选择决策应用于修剪所有早期层的KV缓存，并在后续层中使用
- 这种策略最小化了剪枝开销，防止浅层次优选择影响后续层计算，最大化了KV缓存减少

## 5. 数据与实验

**数据集**：
- 在四个代表性的LVLMs上评估：Qwen2.5-VL、InternVL 2.5、DeepSeek-VL2和LLaVA v1.5
- 使用20个多样化的基准测试，涵盖：
  - 视觉定位：RefCOCO、RefCOCO+、RefCOCOg
  - 一般推理：SEEDBench、MMBench、MMStar、MME
  - 幻觉评估：POPE、HallusionBench
  - 现实世界理解：RealWorldQA
  - OCR：TextVQA、AI2D

**对比基线模型**：
- FastV
- Window FastV (W-FastV)
- PDrop
- VScan

**核心实验结果**：
- IVC-Prune减少了约50%的视觉标记，同时保持了≥99%的原始性能
- 在视觉定位任务上显著优于现有剪枝方法（例如，在RefCOCO上保持92.0%准确率，而基线方法为74.4-90.2%）
- 在某些基准测试上甚至实现了性能提升（如SEEDBench达到100.6%相对性能）
- 与其他剪枝方法结合使用时，可以增强其空间推理能力

## 6. 主要贡献

这篇论文对该领域的主要贡献包括：

1. **理论贡献**：首次从理论上分析了LVLMs如何通过RoPE的数学结构隐式建立视觉坐标系，为理解LVLMs的空间推理机制提供了新的见解。

2. **方法贡献**：提出了IVC-Prune，一种新颖的、不需要训练的剪枝策略，同时保留IVC标记和语义相关的标记，并引入了一种可跨LVLM架构和基准测试推广的鲁棒两阶段选择过程。

3. **实践贡献**：通过广泛的实验证明，IVC-Prune可以在减少约50%视觉标记的同时保持≥99%的原始性能，并在视觉定位任务上显著优于现有方法。此外，研究还揭示了早期层剪枝敏感性的根本原因，并提出了有效的解决方案。

4. **领域影响**：IVC标记可以无缝集成到其他剪枝方法中，持续增强其空间推理能力，为LVLMs的高效部署提供了新的可能性。
