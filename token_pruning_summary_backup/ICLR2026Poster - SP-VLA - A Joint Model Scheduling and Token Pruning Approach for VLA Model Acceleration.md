# SP-VLA: A Joint Model Scheduling and Token-Pruning Approach for VLA Model Acceleration 论文总结

## 1. 研究动机
Vision-Language-Action (VLA)模型因其强大的控制能力而受到越来越多的关注，但其高计算成本和低执行频率阻碍了它们在机器人操作和自主导航等实时任务中的适用性。现有的VLA加速方法主要关注结构优化，忽略了这些模型在顺序决策环境中运行的事实，导致顺序动作生成中的时间冗余和视觉输入中的空间冗余未得到解决。此外，现有方法大多针对Vision-Language Model (VLM)架构，而忽略了VLA模型通过与环境持续交互逐步生成动作所带来的额外时间维度特性。

## 2. 解决问题
论文致力于解决的核心科学问题或技术难题是：
1. **时间冗余问题**：如何有效减少VLA模型在顺序决策环境中的时间冗余，利用历史信息支持当前决策
2. **空间冗余问题**：如何有效减少VLA模型在视觉输入中的空间冗余，保留有价值的视觉内容

## 3. 现象分析
作者观察到的有趣现象和深入剖析包括：
1. **行为模式相似性**：VLA模型的行为模式类似于人类，动作可分为"深思熟虑型"(deliberative)和"直觉型"(intuitive)。精确操作如抓取或转向需要深思熟虑，而关键点之间的移动则依靠直觉。
2. **速度与动作类型关联**：高速运动通常是直觉型动作，而低速运动更可能是深思熟虑型动作。
3. **空间感知特性**：VLA模型的空间感知依赖于标记的相对顺序和物体轮廓信息。实验表明，扰乱标记排列或激进剪除背景标记会导致VLA任务性能显著下降。
4. **标记冗余性**：随机剪除标记会降低准确性，但不会阻止任务完成，表明许多标记是冗余的。然而，仅根据语义重要性重新排序或剪除标记会导致任务失败。

## 4. 主要方法
论文提出了**SP-VLA**框架，一个通过联合调度模型和剪除标记来加速VLA模型的统一框架。核心机制包括：

### 4.1 动作感知的模型调度机制
- **动作分类**：将VLA动作分为深思熟虑型(_deliberative_)和直觉型(_intuitive_)，前者需要高精度VLA模型处理，后者可由轻量级模型处理
- **轻量级生成器**：基于岭回归(Ridge Regression)设计轻量级动作生成器，利用动作缓存(Action Buffer)高效预测
- **动态切换**：根据历史动作序列和当前运动速度动态切换VLA模型和轻量级生成器
- **触发条件**：当所有速度分量超过预定义阈值_vmin_时使用轻量级模型，同时监控VLA生成动作比例_NG/NA > τ_

### 4.2 空间-语义双感知标记剪除方法
- **语义感知重要性**：通过累积注意力分数(_AccuAttn_)计算标记的语义重要性，选择超过阈值_tks_的标记_Tse_
- **空间感知重要性**：使用Canny算子提取物体轮廓信息，获取空间上相关的标记_Tsp_
- **标记选择**：通过保持原始顺序的并集操作_U(Tse, Tsp)_获得最终选择的标记集合
- **速度自适应剪除**：根据当前速度动态调整标记保留比例，速度越高剪除比例越大，公式为：_ratio = 0.5 × (1 + tanh((v - vp_min)/(vp_max - vp_min)))_

## 5. 数据与实验
### 5.1 数据集
- **LIBERO**：包含四个任务套件(Spatial、Object、Goal和Long)，涵盖130个任务和2000条轨迹，评估模型鲁棒性
- **SimplerEnv**：提供三种设置(GoogleVM、Google-VA和Bridge-VM)，具有颜色、材质、光照和相机位置的变化，用于鲁棒性评估
- **真实机器人实验**：在Franka Research 3机器人上部署SP-VLA，使用CogACT作为基础模型

### 5.2 对比基线模型
- OpenVLA
- SparseVLM
- FoPru + R
- PruMerge + R
- FastVLM + R + S
- VisionZip + R + S

### 5.3 核心实验结果
- **加速效果**：在LIBERO上实现1.5×无损加速，在SimplerEnv上实现2.4×加速
- **性能提升**：平均性能提升高达6%
- **推理效率**：在SimplerEnv上的推理频率和延迟分别提高2.2×和1.4×，在LIBERO上分别提高1.4×和2.2×
- **真实机器人评估**：准确率仅下降1%，同时实现2.5×的端到端加速

## 6. 主要贡献
这篇论文对该领域的主要贡献是：

1. **首创性框架**：首次通过减少时间和空间冗余来加速VLA模型，提出了联合模型调度和标记剪除的统一框架，引导VLA模型关注关键动作和视觉元素以实现最大加速

2. **动作感知调度算法**：提出动作感知的模型调度算法，有效减少VLA模型中的时间冗余。观察到VLA动作序列类似于人类行为，包含直觉型和深思熟虑型动作，将直觉型动作分配给轻量级模型，深思熟虑型动作分配给VLA模型，实现无损、频率自适应的加速

3. **空间-语义双感知标记剪除**：提出空间-语义双感知标记压缩方法，发现VLA模型的空间感知依赖于标记的相对位置和物体轮廓信息，设计结合边缘特征和语义重要性的标记剪除方法，有效减少空间冗余

4. **实际应用价值**：在多个数据集和真实机器人平台上验证了方法的有效性，为VLA模型在实时任务中的应用提供了可行的加速方案
