# 论文总结：Balanced Token Pruning: Accelerating Vision Language Models Beyond Local Optimization

## 1. 研究动机
- **问题背景**：大型视觉语言模型(LVLMs)通过将图像编码成数千个token在多模态任务中表现出色，但大量图像token导致显著计算开销，动态高分辨率输入进一步增加这一负担。
- **应用痛点**：在边缘应用（如应急监控、物流、智能家居）中，模型部署在无人机和无人驾驶车辆等设备上，受限于内存和严格延迟要求。
- **现有局限**：现有token剪枝方法（基于注意力分数或图像token多样性）往往忽略剪枝对当前层输出（局部）和后续层输出（全局）的联合影响，导致次优剪枝决策。

## 2. 解决问题
- **核心科学问题**：如何在视觉token剪枝中平衡局部优化（保持当前层输出）和全局优化（保持后续层输出）。
- **技术难题**：如何确定适当的剪枝层，实现全局最优性而非局部最优性。
- **研究目标**：设计一种能够同时考虑当前层和后续层影响的token剪枝方法，超越现有方法的局部优化限制。

## 3. 现象分析
- **关键发现1**：不同层中从文本token接收更高注意力的图像token的空间分布存在差异（图1），表明仅基于当前层进行剪枝会忽略对后续层的影响。
- **关键发现2**：基于注意力的方法在早期剪枝层保持输出相似性良好，但在深层中误差累积；基于多样性的方法在初始层不保持输出相似性，但在后期剪枝阶段实现更好一致性（图2）。
- **深入剖析**：基于注意力的剪枝方法仅专注于优化当前剪枝层，忽略对后续层影响；基于多样性的方法忽略保持当前层输出质量。
- **重要观察**：图像token的表示在某些层会发生显著变化，这些层之后模型往往会为图像token分配更多注意力。

## 4. 主要方法
- **提出方法**：**平衡的token剪枝(Balanced Token Pruning, BTP)**，一种即插即用的视觉token剪枝方法。
- **核心机制**：
  - **多阶段剪枝策略**：将剪枝过程划分为多个阶段，使用小型校准集确定阶段划分。
  - **动态目标平衡**：在早期阶段（保留更多token）强调基于多样性的目标，在后期阶段（保留较少token）优先考虑基于注意力的目标。
  - **局部-全局目标函数**：结合注意力优化（保持当前层输出）和多样性优化（保留对后续层重要的token），通过权重系数λ调节平衡。
  - **注意力再平衡操作**：通过过选择和位置再平衡缓解位置编码偏差。
  - **空间初始化策略**：基于空间位置信息解决Max-Min多样性问题(MMDP)的计算效率问题。
  - **任务无关剪枝层选择**：基于图像token隐藏状态变化确定剪枝层，而非依赖验证性能和手动设置。

## 5. 数据与实验
- **数据集**：
  - 模型：LLaVA-v1.5-7B、LLaVA-v1.5-13B、LLaVA-v1.6-7B、Qwen2.5-VL-7B
  - 基准：GQA、MMB、MME、POPE、SQA和MM-VeT
  - 校准集：64个随机采样的LLaVA-655k数据集实例

- **基线模型**：
  - VTW：在特定transformer层删除所有图像token
  - PyramidDrop (PDrop)：在整个transformer中执行阶段性剪枝
  - FastV：通过过滤层后剪枝低注意力分数的视觉token
  - DivPrune：基于视觉多样性过滤token

- **核心实验结果**：
  - 在78%的压缩率下平均保留原始模型96.7%的性能
  - 在22%的token保留率下平均保留98%的原始性能
  - 在大多数基准任务上优于所有对比方法
  - 显著减少推理延迟和KV缓存内存使用
  - 在不同压缩比率下保持稳定性能

## 6. 主要贡献
- **新方法**：提出BTP方法，首次平衡局部和全局目标的视觉token剪枝方法。
- **新发现**：揭示现有token剪枝方法的根本局限性（注意力方法追求局部最优，多样性方法忽略局部约束）。
- **新洞察**：阐明剪枝对当前层和后续层的不同影响规律，以及层间token表示的变化模式。
- **新策略**：提出基于图像token编码过程的任务无关剪枝层选择策略。
- **高效实现**：提出空间位置信息初始化策略，有效解决MMDP计算效率问题。
- **广泛有效性**：在多种LVLM架构和规模上验证方法的有效性，实现即插即用的部署特性。
