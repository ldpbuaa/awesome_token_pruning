# 论文深度总结：Why 1 + 1 < 1 in Visual Token Pruning: Beyond Naïve Integration via Multi-Objective Balanced Covering

## 1. 研究动机

- **现有痛点**：现有视觉令牌剪枝(Visual Token Pruning)方法针对提示对齐(Prompt Alignment, PA)和视觉保持(Visual Preservation, VP)采用静态策略，忽视了这两个目标在不同任务中相对重要性的变化，导致性能不一致。
- **关键问题**：多目标方法集成VP和PA并未表现出预期的优越性，反而与单目标方法相比性能不稳定，引发了一个根本性问题："集成不同目标是否具有基本优势？"
- **背景**：多模态大语言模型(MLLMs)在处理视觉输入时面临计算开销大的问题，因为视觉数据的空间冗余性高，通常需要编码为大量令牌，而视觉令牌剪枝是一种加速MLLMs的有效方法。

## 2. 解决问题

- **核心科学问题**：如何量化提示对齐(PA)和视觉保持(VP)这两个目标之间的权衡关系？
- **技术难题**：如何根据不同的提示-视觉耦合模式动态调整这两个目标的相对重要性，使剪枝方法能够适应不同任务场景？
- **研究目标**：设计一个能够根据任务特性自适应平衡PA和VP的视觉令牌剪枝方法，在各种场景下都能保持良好性能。

## 3. 现象分析

- **关键发现**：作者在多个视觉-语言基准上观察到两种不同的提示-视觉耦合模式：
  - **弱耦合**：提示和视觉令牌之间的Hausdorff距离较大(如TextVQA, POPE任务)
  - **强耦合**：提示和视觉令牌之间的Hausdorff距离较小(如MMB, VizWiz任务)
- **性能差异**：在不同耦合模式下，剪枝目标的相对有效性不同：
  - 弱耦合时，提示对齐(PA)更重要
  - 强耦合时，视觉保持(VP)更重要
- **现有方法局限**：多目标方法忽略了这种变化，使用恒定策略组合VP和PA，导致在某些任务上比单目标方法表现更好，而在其他任务上表现更差(如图1a所示)。

## 4. 主要方法

- **理论框架**：基于Hausdorff距离推导了视觉令牌剪枝的第一个闭式误差边界，揭示了PA和VP之间的内在权衡关系。
- **核心算法**：多目标平衡覆盖(Multi-Objective Balanced Covering, MoB)，其核心机制包括：
  - 将视觉令牌剪枝重新表述为双目标覆盖问题
  - 将保留的令牌划分为两个不相交子集：提示中心集($S_p$)和视觉中心集($S_v$)
  - 通过两种贪婪的半径交易策略将目标实现的权衡转化为预算分配问题
- **具体实现**：
  - 提示中心集选择：采用k-fold最近邻覆盖(k-fold NN covering)选择与提示令牌最相关的视觉令牌
  - 视觉中心集选择：使用最远点采样(Farthest Point Sampling)在剩余令牌中选择分布最均匀的视觉令牌
- **性能保证**：提供可证明的性能边界，与输入视觉令牌数量呈线性可扩展性。

## 5. 数据与实验

- **数据集**：在14个公共基准上进行了实验，包括LLaVA-1.5-7B、Video-LLaVA-7B、LLaVA-Next-7B、Qwen2-VL等，涵盖视觉理解、视觉问答、视觉语言推理等多种任务。
- **基线模型**：
  - 单目标方法：FastV(VP)、SparseVLM(PA)
  - 多目标方法：MustDrop(VP+PA)
  - 其他相关方法：ECCV'24的FastV、ICML'25的SparseVLM等
- **核心实验结果**：
  - MoB在LLaVA-1.5-7B上仅使用11.1%的原始视觉令牌就保留了96.4%的性能
  - MoB将LLaVA-Next-7B的推理速度提高了1.3-1.5倍，同时 negligible性能损失
  - 在各种剪枝率下，MoB在10个公认基准上始终优于其他方法(如图1c所示)
  - 在Qwen2-VL和Video-LLaVA上的评估表明MoB可以无缝集成到先进的MLLMs中

## 6. 主要贡献

- **理论贡献**：
  - 提出了视觉令牌剪枝的第一个闭式误差边界，表征了两个目标对保持质量的贡献
  - 量化了目标之间的权衡，确定了在固定预算和提示-视觉耦合下它们的最优实现水平
- **方法贡献**：
  - 提出了无需训练的视觉令牌剪枝方法MoB，通过两种贪婪的半径交易策略将目标实现的权衡减少为预算分配问题
  - 提供了可证明的性能保证和线性可扩展性
- **实验贡献**：
  - 在14个公共基准上进行了大量实验，证明了MoB的优越性
  - 展示了MoB能保留LLaVA-1.5-7B 96.4%的性能，同时将视觉令牌减少88.9%
  - 证明了MoB可以轻松集成到先进的MLLMs(如LLaVA-Next和Qwen2-VL)中
