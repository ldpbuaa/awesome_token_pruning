# 论文总结：Attribution-Driven Adaptive Token Pruning for Transformers

## 1. 研究动机

- **计算成本问题**：Transformers模型在处理长输入序列时计算成本过高，显著增加了训练和推理时间，限制了其在实时或边缘设备上的部署。
- **现有方法局限性**：大多数token剪枝方法忽略了序列在长度和复杂性方面的关键差异，导致压缩效率次优。
- **注意力值不准确**：现有方法主要依赖注意力分数评估token重要性，但注意力值不能准确反映每个token对文本理解的实际贡献。
- **静态配置问题**：许多剪枝策略采用固定或全局学习阈值，无法适应不同序列复杂性的变化。

## 2. 解决问题

- **核心科学问题**：如何更准确地评估token的重要性，以保留最具信息量的token，同时根据每个输入序列的复杂性动态调整剪枝配置。
- **技术难题**：在减少计算成本的同时保持或提高模型性能，解决现有方法在处理不同复杂度序列时出现的欠剪枝或过剪枝问题。

## 3. 现象分析

- **注意力值局限性**：实验表明，基于注意力的方法在评估token重要性时表现不佳，存在注意力分数与token实际语义贡献之间的差异。
- **序列复杂性差异**：数据集中不同序列在长度和复杂性上存在显著变化（如图2和图3所示），需要自适应处理。
- **固定阈值缺陷**：使用固定或全局学习阈值的剪枝框架无法适应序列长度和复杂性的变化，导致性能下降。
- **模型表示对齐挑战**：剪枝过程中token移除会导致信息损失，造成学生模型和教师模型之间的表示不对齐。

## 4. 主要方法

### AD-TP (Attribution-Driven Adaptive Token Pruning)

1. **归因感知自适应token保留器**：
   - **ARP模块**：通过特殊分类token `[CLS]` 的嵌入估计序列复杂度并确定保留率 ρr
   - **TSP模块**：评估嵌入序列中每个token的重要性，分配显著性分数 S

2. **基于Integrated Gradients (IG)的token显著性估计**：
   - 使用IG计算token重要性，通过集成从基线输入到实际输入的梯度来计算归因分数
   - 公式：$IG_i^c = \int_{\alpha=0}^{1} \nabla_{\hat{E}} F^c(\alpha \cdot \hat{E} + (1-\alpha) \cdot E_{baseline}) \cdot (\hat{E} - E_{baseline}) \, d\alpha$

3. **基于教师模型和自监督的联合监督**：
   - 结合教师模型和学生模型的归因分数构建统一监督信号
   - 采用双重归一化策略确保特征对齐
   - 引入自监督目标增强学生模型的token级推理能力

4. **整体训练目标**：
   - 结合分类损失、基于logits的蒸馏损失、归因损失和token保留器监督
   - 使用超参数α、β和γ控制各损失项的贡献

## 5. 数据与实验

### 数据集
- **GLUE基准测试**：包含8个NLP任务
- **长文本数据集**：SQuAD v2.0和20News

### 基线模型
- **注意力方法**：PoWER-BERT、LAT、LTP、SpAtten
- **预测方法**：Transkimmer、ToP
- **蒸馏方法**：DistilBERT
- **结构化剪枝**：CoFi

### 实验结果
- **GLUE基准**：
  - AD-TP将FLOPs平均减少7.8%，同时提高性能0.6%
  - AD-TP6（6层Transformer）平均减少FLOPs 7.02×
  - AD-TP12（12层Transformer）平均减少FLOPs 7.37×

- **长序列任务**：
  - SQuAD v2.0：AD-TP6实现76.2的F1分数，FLOPs减少5.07×
  - 20News：AD-TP6达到87.3%的准确率，FLOPs减少8.74×

- **对比实验**：
  - 与DistilBERT相比，AD-TP实现更高的压缩率（BERT6为8.7× vs 2.0×）
  - 与CoFi相比，AD-TP在相似或更高压缩比下表现更好
  - 在不同FLOPs预算约束下，AD-TP consistently优于LTP和ToP基线

## 6. 主要贡献

1. **AD-TP方法**：引入轻量级自适应token保留器，根据序列复杂度动态确定保留率，显著降低计算成本。

2. **基于归因的token重要性估计**：利用集成梯度(IG)量化模型预测与序列特征之间的关系，提供更准确的token重要性评估。

3. **双重归一化知识蒸馏框架**：对齐教师模型和学生模型之间的归因特征，并融入自监督增强学生模型的token级推理能力。

4. **全面实验验证**：在GLUE、SQuAD和20News上的实验表明，AD-TP在准确性和计算效率上都优于最先进的token剪枝和模型压缩方法。

5. **解决关键挑战**：
   - 克服了对注意力分数的依赖，提供了更准确的token重要性评估
   - 通过动态调整剪枝决策，解决了静态配置无法适应序列复杂性变化的问题
