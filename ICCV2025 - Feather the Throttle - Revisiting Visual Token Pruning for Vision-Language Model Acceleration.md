# 论文总结：Feather the Throttle: Revisiting Visual Token Pruning for Vision-Language Model Acceleration

## 1. 研究动机
- **背景痛点**：视觉语言模型(VLMs)将大型语言模型(LLMs)与视觉编码器结合，但使用图像块作为标记(patch features)带来了计算效率低下的问题。图像被划分为许多小块(patch)，由于Transformer中注意力操作的二次方复杂度，大量标记显著增加了计算需求。
- **现有方法局限**：现有加速方法(如FastV)通过在浅层语言模型中剪枝视觉标记来提高效率，但这些方法在大多数视觉语言任务上表现良好，但在某些以视觉为中心的任务(如定位任务)上表现较差。
- **研究动机**：作者希望更好地理解这些压缩视觉信息的加速VLMs的视觉能力，特别关注FastV方法，探究为什么这些方法在评估视觉能力的任务上仍能保持高性能，以及是否存在更 demanding 的视觉任务会使这些方法失效。

## 2. 解决问题
- **核心科学问题**：探究视觉标记剪枝加速方法在视觉语言模型中的有效性，特别是为什么这些方法在大多数任务上表现良好，但在某些以视觉为中心的任务(如定位任务)上表现严重下降。
- **技术难题**：解决早期视觉标记剪枝方法中存在的偏向性问题，即剪枝标准在早期层中偏向于选择图像底部的标记，导致对需要精细视觉信息的任务(如定位)性能大幅下降。

## 3. 现象分析
- **关键现象1**：早期视觉标记剪枝对大多数视觉语言任务影响很小，但对TextVQA任务造成显著性能下降，对定位任务(如RefCOCO, RefCOCO+, RefCOCOg和OCID-Ref)造成严重性能下降(下降幅度分别为88.9%, 88.9%, 91.0%和86.0%)。
- **关键现象2**：定位性能随着剪枝比例的增加大致线性下降到零，这与大多数其他任务形成鲜明对比，后者的性能在大量标记被移除后基本保持不变。
- **关键现象3**：早期层(如K=3)的剪枝标准导致保留的标记集中在图像底部，当剪枝比例为75%时，平均采样标记位置位于图像80.7%的位置。这种偏向性随着剪枝层加深而减少。
- **关键现象4**：即使在早期层使用次优剪枝标准，大多数任务仍能保持高性能，这表明许多基准测试不需要详细的视觉信息就能准确回答问题。

## 4. 主要方法
- **FEATHER方法**：Fast and Effective Acceleration wiTH Ensemble cRiteria，一种改进的视觉标记剪枝策略。
- **核心机制**：
  1. **解决位置偏差问题**：通过从注意力机制中移除RoPE(Rotary Position Embedding)来消除长期衰减效应，使注意力分数能更有效地用于选择与文本指令相关的标记。
  2. **多阶段剪枝策略**：在早期层进行轻度剪枝并加入均匀采样以确保整个图像的良好覆盖，在更深层的注意力机制能更好地区分重要标记时进行更广泛的剪枝。
  3. **早期均匀采样**：在早期剪枝阶段加入均匀采样标记，确保图像所有区域都有覆盖。
- **类比说明**：这种方法类比于赛车手在转弯时如何"轻踩油门"(feather the throttle)，在转弯开始时逐渐踩下油门以保持抓地力，一旦通过拐点再更激进地加速。

## 5. 数据与实验
- **使用的数据集**：
  - **定位任务**：RefCOCO, RefCOCO+, RefCOCOg和OCID-Ref
  - **开放式视觉问答**：VizWiz, VQAv2, GQA和TextVQA
  - **挑战集**：VSR, TallyQA, POPE和AI2D
- **对比的基线模型**：
  - 原始FastV方法(ω_original)
  - 移除RoPE的改进方法(ω-R)
  - 均匀采样方法(ω_uniform)
  - 基于局部密度的KNN方法(ω_KNN)
  - ω-R与均匀采样的组合方法(ω-R + ω_uniform)
- **核心实验结果**：
  - 通过移除RoPE，在K=3时定位任务平均性能提升183%，在K=8时提升17%
  - FEATHER方法在保持相同计算节省的情况下，相比原始加速方法在以视觉为中心的定位基准测试上实现了超过5倍的性能提升
  - FEATHER方法只保留了3.3%的视觉标记用于LLM的后半层，同时实现了显著的性能提升

## 6. 主要贡献
- **揭示早期剪枝的不一致影响**：在12个基准测试上的评估表明，早期视觉标记剪枝对大多数任务性能影响最小，但对TextVQA造成显著下降，对定位任务造成严重下降。
- **分析性能不佳的原因**：发现用于选择重要标记的基于注意力的标准在早期层中由于RoPE的长期衰减特性而对底部图像标记产生偏向性。
- **识别基准测试在评估标记剪枝方面的局限性**：发现即使使用次优剪枝标准并在LLM中转移信息之前移除被剪枝的标记，性能仍然保持强劲，表明许多基准测试无法评估精细的视觉能力。
- **改进基于注意力的标准**：通过简单地移除RoPE，证明注意力可以成为选择标记的有效标准，并通过结合均匀采样和在更深层进行更广泛剪枝等简单策略可以进一步提高性能。
